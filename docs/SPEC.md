# Specification: Mini System Monitor

**バージョン**: 0.1.10
**内部バージョン**: 0.1.10.0

## 1. 概要
低リソース環境（Windows）で動作する、極限まで軽量なシステムモニタ。
常に最前面に表示され、システムリソースをリアルタイム（1秒間隔）で監視する。

## 2. 技術スタック
- **Prototype:** React, Tailwind CSS, Framer Motion
- **Production:** Rust, egui, eframe (0.35+), sysinfo (0.38+), chrono, serde, serde_json, common_lib (ローカル共有クレート)

## 3. 外観仕様
- **サイズ:** 1100px x 32px (横長バー)
- **装飾:** タイトルバー非表示 (Borderless)
- **背景:** 半透明 (Alpha: 200/255)
- **フォント:** 視認性の高いサンセリフ
- **配置:** 常に最前面 (Always on Top)

## 4. 機能仕様
- **CPU監視:** `sysinfo` を使用し、全コアの平均使用率を表示。
- **CPU温度:** センサーから温度を取得し表示。
- **メモリ監視:** 使用済みメモリ / 全メモリ のパーセンテージを表示。
- **ネットワーク監視:** 合計アップロード/ダウンロード速度を表示（上り `^` / 下り `v`）。
- **ディスク容量監視:** システムディスクの使用量と全容量を表示 (例: 123GB/512GB)。
- **ディスクI/O監視:** 全プロセスのディスク使用量を合算して算出した、リアルタイムの読み込み (R) / 書き込み (W) 速度を表示.
- **バージョン表示:** IO表示と時計表示の間にアプリケーションのバージョン (`vX.Y.Z` 形式) を表示。
- **時計表示:** 右端に現在日時を `yyyy/mm/dd(Day) hh:mm:ss` 形式で表示。
- **移動:** 左端のグリップアイコン `=` をドラッグしてウィンドウを移動。
- **終了:** 右端の「×」ボタンでアプリケーションを終了。
- **位置保存:** 終了時のウィンドウ位置を自動保存し、次回起動時に復元。
- **二重起動防止:** 共有ライブラリ `common_lib`（Windows の名前付きミューテックスの安全なラッパー）を利用した多重起動防止機能。既にアプリが起動している場合は、新しく起動されたプロセスが即座に正常終了する。
- **視認性:** テキスト選択を無効化し、誤操作を防止。

## 5. 最適化
- **描画更新:** 1秒間隔に固定し、CPU負荷を最小化。
- **レイアウト安定化:** 各項目に固定幅を割り当て、数値変動によるガタつきを防止。
- **API互換性:** `sysinfo` 0.38 および `eframe` 0.35 の最新APIに準拠。
- **最小限のシステムスキャン:** 1秒ごとの更新時、システム情報全体のフルスキャン（`refresh_all`）を排除し、必要な CPU・メモリ情報のみをリフレッシュ。また、ディスク I/O 取得のためのプロセス情報更新においても、`ProcessRefreshKind` を用いてディスク使用量以外の無駄なデータ取得をスキップ。
