# Todo List: Mini System Monitor (MiSysMon)

本プロジェクトの開発ステータス、直近のタスク、および将来的なバックログ（機能拡張のロードマップ）を管理するドキュメントです。

---

## 1. 実装済みの機能 (Done)

- [x] **基本UIフレームワークの構築**:
  - `egui` および `eframe` を使用したスリムな横長バー (1100px x 32px) のデスクトップウィンドウ表示。
  - タイトルバー非表示 (Borderless)、背景透過 (Alpha: 200/255)、常に最前面 (Always on Top)、およびリサイズ禁止の設定。
- [x] **リアルタイム・システムメトリクス取得**:
  - **CPU**: `sysinfo` を用いた全コアの平均使用率のパーセンテージ取得。
  - **CPU温度**: センサー一覧から「CPU」「Core」「Package」などのラベル付きセンサーを検知して温度を取得。検知できない場合は全センサーの最高温度を採用するフォールバック処理。
  - **メモリ**: 使用済みメモリ割合のパーセンテージ取得。
  - **ネットワーク**: インターフェースごとの受信・送信トラフィックを合算し、前回の更新との差分からアップロード/ダウンロード速度を算出して表示。
  - **ディスク**: 全ディスクの合計空き容量・総容量の表示 (GB単位)。
  - **ディスクI/O**: 全プロセスのディスク読み込み・書き込みバイト数を合算し、現在のI/O速度（R/W）を算出して表示。
- [x] **診断・デバッグ用機能**:
  - 起動時に検出されたすべての温度センサー情報（ラベルと温度）を `sensors_debug.log` に書き出す診断ログ出力機能。
- [x] **ユーティリティ・利便性機能**:
  - 左端のドラッググリップ（`=`）を用いた、タイトルバーなしウィンドウのドラッグ移動。
  - 終了時のウィンドウの左上座標を自動で JSON 永続化保存し、次回起動時にその座標に復元する機能。
  - 右端の終了ボタン（`×`）の設置。
  - 右端固定の時計表示（点滅しない固定幅レイアウト `yyyy/mm/dd(Day) hh:mm:ss`）。
  - テキスト選択の無効化（誤クリック・ドラッグによるガタつき防止）。
- [x] **多重起動防止機能**:
  - 共有ライブラリ `common_lib` の名前付きMutexによる起動制御。すでに同一のキーを持つインスタンスが動いている場合は、新しいプロセスは何もせず即終了する。
- [x] **ネットワーク初回起動時の表示スパイク修正**:
  - `networks` の生成時に初期の累積通信量をあらかじめ取得して baseline とすることで、起動後最初の1秒目の速度が極端に巨大な値（OS起動時からの総累積通信量）として表示されるバグを修正。
- [x] **マルチモニター環境ウィンドウ位置の画面内検証（Win32 API）**:
  - Win32 API の `MonitorFromPoint` をインポートし、保存された座標が現在の有効なモニターの表示範囲外にある場合はデフォルト位置に配置するフォールバックロジックを実装。
- [x] **温度センサー取得処理のクリーンアップ・診断機能の強化**:
  - センサー数が0個の場合に、起動時生成される `sensors_debug.log` 内に「温度取得には管理者権限での起動が必要な可能性がある」旨の案内を自動で出力するように改善。
- [x] **ログファイルのローテーション機能**:
  - 起動時に古い `sensors_debug.log` ファイルを `sensors_debug.log.1`, `.2`, `.3` へ順番にリネームシフトする簡易ローテーションロジックを実装（最大3世代のバックアップを維持）。
- [x] **ウィンドウのオートスナップ（吸着）機能**:
  - ウィンドウをデスクトップの画面端（特に上下左右の境界線）にドラッグした際、15px以内に近づくと自動的にピタッと吸着する機能を Win32 API を用いて実装。
- [x] **リソース使用率のグラフ（ミニトレンドライン）表示**:
  - CPU・メモリ使用率の隣に、直近30秒間の推移を高さ12pxのミニスパークライン（折れ線グラフ）としてリアルタイム描画する機能を実装。
- [x] **設定カスタマイズ画面の追加**:
  - グリップ（`=`）のダブルクリックまたはメインパネル全体の右クリックで、別ウィンドウ（サブビューポート）の設定ダイアログを表示する機能を実装。
  - 更新間隔（0.5秒〜5.0秒）、ウィンドウ不透明度（30%〜100%）、オートスナップの有効/無効、各項目の表示トグル（CPU、メモリ、ネットワーク、ディスク、ディスクIO、バージョン、時計）がカスタマイズ可能。
  - 表示トグルに合わせてメインウィンドウの幅を動的に自動リサイズし、不要な余白を排除するプレミアムなレイアウト調整を実装。
