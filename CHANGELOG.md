# Changelog

## [2026-06-25]
### Added
- **自動ドキュメント作成・更新ルールの追加:** AIエージェント用の個別指示書 `.agents/AGENTS.md` を新規作成。コード変更時に `CHANGELOG.md` や `SPEC.md` などの関連ドキュメントを自動で走査・更新するプロセスを規定しました。
- **自動化スクリプトの追加:** `scripts/` ディレクトリを新設し、Rustプロジェクト用のバージョン更新・ドキュメント（`docs/SPEC.md`, `docs/TEST_REPORT.md`）同期スクリプト `bump-version.ps1` を追加しました。これに伴い、`SPEC.md` および `TEST_REPORT.md` にバージョン管理用のメタデータヘッダーを追加しました。

### Optimized
- **内部ドキュメントの整理:** プロジェクトルートに散らばっていた内部用設計・検証ドキュメント（`SPEC.md`, `DIAGRAM.md`, `FOOTPRINTS.md`, `TEST_REPORT.md`）を新設した `docs/` ディレクトリに移動し、プロジェクトルートを整理しました。
- **テンプレートガイドの改訂:** `docs/project_template_guide.md` をTauri v2用の構成から、eframe/egui等を用いた純粋なRustデスクトップアプリ開発向けの構成へ完全に改訂・最適化しました。

## [2026-06-24]
### Fixed
- **ディスクI/O速度の算出バグ修正:** 各プロセスの `read_bytes` / `written_bytes` は `sysinfo` 側ですでに差分（前回の更新以降のバイト数）として返されているため、自前での二重差分計算（`prev_disk_read` 等の引き算）を行っていたバグを修正。これによりディスクI/Oの表示が正しく更新されるようになりました。

### Optimized
- **`sysinfo` 更新処理の軽量化による CPU 負荷削減:**
  - 1秒間隔の `refresh_all()` を廃止し、必要な `refresh_cpu()` および `refresh_memory()` のみの呼び出しに変更。
  - ディスクI/Oを計算するためのプロセス情報更新において、`ProcessRefreshKind::new().with_disk_usage()` を指定することで、余分な情報（CPU、メモリ、環境変数など）の取得をスキップし、ディスク使用量のみを更新するように最適化。
  - アプリ起動時の初期化処理を `System::new()` に変更し、起動時のオーバーヘッドを最小限化。
  - **診断ログ `sensors_debug.log` の出力先変更による安全性向上:** カレントディレクトリが汚れるのを防ぎ、スタートアップ等での起動時の書き込み権限エラーを避けるため、出力先を設定ファイルと同じ `%APPDATA%\Mini System Monitor` ディレクトリに変更しました。

### Added
- **二重起動防止機能の追加 (Windows):** Windows 名前付きミューテックスを用いた、アプリケーションの多重起動防止機能を実装。既にアプリが起動している場合は即座に正常終了します。
- **MITライセンスファイルの追加:** プロジェクトルートに正式な `LICENSE` ファイルを追加。
- **ドキュメントの強化 (README.md / README.ja.md):**
  - Windows起動時の自動実行（常駐化）設定の手順を追加。
  - 設定データ（ウィンドウ位置など）の物理的な保存場所と、不具合発生時のリセット手順を追加。

## [2026-05-07]
### Fixed
- **CPU温度取得ロジックの更なる堅牢化:** AMD環境等で使われる Tctl/Tdie や Thermal ラベルの検索を追加。データが取得できない場合に 0.0 ではなく "--" と表示されるよう UI を改善。
- **CPU温度取得ロジックの改善:** 一部の環境で0度と表示される問題に対応。明示的なCPUラベルが見つからない場合、全センサー中の最大値を採用するフォールバック処理を追加。

## [2026-04-16] - Final Release Candidate
### Added
- **ディスクI/O表示:** 全プロセスの合計から算出されるリアルタイムのRead/Write速度表示を追加。
- **ネットワーク速度表示:** 累計ではなく、1秒あたりの転送速度を表示するように改善。
- **ウィンドウ位置の保存:** アプリ終了時の位置を保存し、次回起動時にその位置で開く機能を実装。
- **CPU温度表示:** `sysinfo::Components` を使用したCPU温度取得機能。
- **リリース最適化:** `Cargo.toml` にサイズ最適化とストリップ設定を追加。

### Fixed
- **コンソールウィンドウの非表示:** Windows で実行時に背後で DOS 窓が表示されないよう `#![windows_subsystem = "windows"]` を追加。
- **ウィンドウサイズの厳格な固定:** 再起動時に高さが広がる問題を、最小/最大サイズの固定と起動時の強制リサイズにより解消。
- **依存関係の解決:** `serde` 等のバージョンを明示的に指定し、ローカル環境でのビルドエラーを解消。
- **文字化け対策:** 特殊記号（↑↓☷）を標準的な文字（^v=）に置き換え、フォント環境に依存しない表示を実現。
- **API適正化:** `eframe` 0.27 および `sysinfo` 0.30 の最新仕様に準拠。

### Removed
- **表示トグル機能:** 動作が不安定だったため、表示/非表示（トグル）機能を削除。
